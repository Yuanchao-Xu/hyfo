Welcome

hyfo is an R package which can be used as a tool for hydrological and forecasting. There are several tools in R including data processing and data visualization. For hydrological and hydraulic modellers, hyfo can be a good preprocessing and postprocessing tool for you.

hyfo has been tested stable on windows platform.

Installation

You can directly download from the link above, and install in an R IDE e.g. Rstudio. Or use the

Simplest way to install the latest version.

  install.packages('devtools')
  # Ignore the warning that Rtool is not installed, unless you want other function from devtools.
  # If "devtools" is already installed in your computer, you can directly run the following code.
  devtools::install_github('Yuanchao-Xu/hyfo')
