Welcome

hyfo is an R package which can be used as a tool for hydrology and climate forecasting. There are several tools in R including data processing, data visualization and data analysis. For hydrological and hydraulic modellers, hyfo can be a good preprocessing and postprocessing tool for you.

hyfo has been tested stable on windows platform.

Installation

You can directly download from the link above, and install in an R IDE e.g. Rstudio. Or use the Simpler way to install the latest version.

If meet with some error during the installation, most probably is because of the conflicts between packages (e.g., "Rcpp" and "sp"), you can just follow the error message and re-install the required packages, they are all on CRAN, which is very convenient.

  install.packages('devtools')
  # Ignore the warning that Rtool is not installed, unless you want other function from devtools.
  # If "devtools" is already installed in your computer, you can directly run the following code.
  devtools::install_github('Yuanchao-Xu/hyfo')
